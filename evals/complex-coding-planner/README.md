# complex-coding-planner evals

这些 fixture 用于人工或外部评估器检查 planner 行为。

覆盖重点：

- direct / managed / needs-clarification 分级。
- managed 任务生成 `execution-plan.md`。
- 环境、Git、验证和工具清单。
- Plan Quality Gate、Plan Self-Review、Readiness Gate、Plan Approval。
- Execution Contract、Goal Condition、Planning Loop Protocol、Executor Work Loop。
- 研究 findings 落盘、重大决策前重读计划、Plan Amendment Gate。
- Readiness 后停止等待用户批准，不直接实现。
- 提交授权必须单独记录。
